# Version Control with Git & GitHub

## Objective

Learn the fundamentals of Git and GitHub for version control and collaboration.

## Tools Used

* Git
* GitHub
* Git Bash

## Commands Practiced

* git init
* git status
* git add .
* git commit -m "Initial commit"
* git log --oneline
* git push

## Learning Outcomes

* Initialized a Git repository
* Tracked and staged files
* Created commits
* Managed a GitHub repository
* Pushed local changes to a remote repository
